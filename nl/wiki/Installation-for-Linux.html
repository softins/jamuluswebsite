Installatie voor Linux

Zorg dat je de aan de slag pagina hebt gelezen.

Upgraden? Misschien wil je eerst een back-up van je configuratie maken.

Een server installeren? Lees eerst de handleiding voor serverbeheerders.

Debian en Ubuntu

Gebruik de officiële repository (aanbevolen)

We bieden een repository aan voor op Debian en Ubuntu gebaseerde distributies die automatische updates naast jouw andere systeemupdates mogelijk maakt. Om de repository te gebruiken:

  1. Open een consolevenster (CTRL+ALT+T zou moeten werken)
  2. Alleen Ubuntu - Schakel de Ubuntu “universe”-repository in (je kunt de GUI-gebaseerde benadering of CLI-gebaseerde benadering gebruiken).
  3. Download het repository setup script:
    curl https://raw.githubusercontent.com/jamulussoftware/jamulus/main/linux/setup_repo.sh > setup_repo.sh
  4. Maak het script uitvoerbaar: 
    chmod +x setup_repo.sh
  5. Voer het setup script uit en installeer Jamulus:
    sudo ./setup_repo.sh && sudo apt install jamulus

De .deb-bestanden handmatig installeren

  1. Download Jamulus voor jouw architectuur: amd64, armf of arm64
  2. Alleen Ubuntu - Schakel de Ubuntu “universe”-repository in (je kunt de GUI-gebaseerde benadering of CLI-gebaseerde benadering gebruiken).
  3. Werk apt bij door een consolevenster te openen (CTRL+ALT+T zou moeten werken) en typ: sudo apt-get update
  4. Ga naar de locatie waar je het installatieprogramma hebt gedownload en dubbelklik erop of gebruik de opdrachtregel: sudo apt install ./jamulus_3.11.0_ubuntu_amd64.deb (of voor Raspberry Pi etc. zoals hierboven)
  5. Eenmaal geïnstalleerd, dan kun je het bestand verwijderen en alle consolevensters sluiten.

Upgraden: Als je Jamulus moet upgraden naar een nieuwere versie en Jamulus handmatig moet installeren, download dan gewoon het nieuwe .deb-bestand en installeer het opnieuw zoals hierboven.

Hardware configuratie

De Jamulus-client moet verbinding maken met een actieve JACK server om te kunnen starten. Je moet uitzoeken welke de low-latency sound server is voor jouw distributie. - Als jouw distributie JACK gebruikt, zie dan hoe je JACK configureert met QjackCtl. - Als jouw distributie PipeWire gebruikt, zie dan hoe je PipeWire configureert.

Configureer JACK met QjackCtl

Om een JACK-server te draaien, is de aanbevolen methode om QjackCtl te gebruiken.

  1. Start QjackCtl. Je ziet het Qt JACK Control utility hoofdvenster.
  2. Configureer jouw geluidsapparatuur als volgt:
  • Stel de audio Interface in op degene die je wilt (er kunnen er meerdere in de lijst staan - kies de juiste, want deze kan niet worden gewijzigd zonder Jamulus en JACK te stoppen).
  • Stel de Sample Rate in op 48000.
  • Stel de Frames/Period in op 128 en Periods/Buffer op 2 om mee te beginnen.

Start JACK opnieuw op om de nieuwe instellingen van kracht te laten worden en ga verder vanaf het gedeelte Start Jamulus hieronder.

Configureer PipeWire

PipeWire gebruikt zijn eigen JACK-server. Wanneer je de Jamulus-client start, start PipeWire automatisch een JACK-server. Er zijn echter een paar parameters die je moet aanpassen. Je moet jouw audio-interface instellen op het “pro-audio”-profiel en de “rate”- en “quantum”-parameters van PipeWire van tevoren definiëren. Je kunt meer informatie over de configuratie vinden in de PipeWire-wiki.

Om jouw systeem in te stellen: 1. Open pavucontrol (de audiomixer) en kies het profiel “pro-audio” voor jouw geluidshardware in het configuratietabblad. 2. Gebruik in een terminal-venster de volgende twee opdrachten om de rate en quantum van PipeWire te forceren:

pw-metadata -n settings 0 clock.force-rate 48000
pw-metadata -n settings 0 clock.force-quantum 128

Ga hierna verder vanaf het gedeelte Start Jamulus hieronder.

Start Jamulus

Start Jamulus nadat JACK of PipeWire is geconfigureerd.

Als je problemen ondervindt, controleer dan eerst de pagina voor probleemoplossing.
