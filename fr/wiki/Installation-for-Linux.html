Installation pour Linux

Assurez-vous d’avoir déjà lu la page Configuration.

Mise à jour ? Vous pouvez d’abord sauvegarder votre configuration.

Installation d’un serveur? Lisez d’abord le guide pour les administrateurs de serveurs

Debian et Ubuntu

Utilisation du dépôt officiel (recommandé)

Nous fournissons un dépôt pour les distributions basées sur Debian et Ubuntu qui permet des mises à jour automatiques en parallèle de vos autres mises à jour du système. Pour utiliser ce dépôt :

  1. Ouvrir un Terminal (CTRL+ALT+T devrait fonctionner)
  2. Ubuntu seulement - Activer le dépôt Ubuntu “universe” (vous pouvez utiliser l’interface utilisateur ou le terminal).
  3. Télécharger le script de configuration du dépôt :
    curl https://raw.githubusercontent.com/jamulussoftware/jamulus/main/linux/setup_repo.sh > setup_repo.sh
  4. Rendre le script exécutable :
    chmod +x setup_repo.sh
  5. Exécuter le script de configuration et installer Jamulus :
    sudo ./setup_repo.sh && sudo apt install jamulus

Installation manuelle des fichiers .deb

  1. Télécharger Jamulus pour votre architecture : amd64, armf ou arm64
  2. Ubuntu seulement - Activer le dépôt Ubuntu “universe” (vous pouvez utiliser l’interface utilisateur ou le terminal).
  3. Mettez à jour apt en ouvrant une fenêtre de console (CTRL+ALT+T devrait fonctionner) et tapez : sudo apt-get update
  4. Naviguez jusqu’à l’endroit où vous avez téléchargé le programme
    d’installation et double-cliquez dessus, ou utilisez la ligne de commande
    sudo apt install ./jamulus_3.10.0_ubuntu_amd64.deb (ou pour Raspberry Pi etc. comme ci-dessus)
  5. Une fois installé, vous pouvez supprimer le fichier et fermer toutes les fenêtres de la console.

Mises à jour : si vous devez mettre à jour Jamulus vers une version plus récente et installer Jamulus manuellement, téléchargez simplement le nouveau fichier .deb et réinstallez comme ci-dessus.

Autres distributions

Pour les installateurs sur d’autres distributions, consultez leurs gestionnaires de paquets et Repology. Si une version à jour de Jamulus n’est pas incluse dans votre distribution, vous pouvez compiler Jamulus à partir des sources. Vous pouvez également utiliser l’un des scripts d’installation.

Configurer votre matériel

Configurer JACK avec QjackCtl

Les clients Jamulus ont besoin de JACK pour fonctionner, mais vous devez d’abord le configurer. La méthode recommandée est d’utiliser QjackCtl.

  1. Lancez QjackCtl. Vous verrez la page principale de l’utilitaire de contrôle JACK Qt
  2. Configurez votre matériel audio comme suit (les paramètres exacts pour JACK dépendront des capacités de votre matériel audio) :
  • Réglez l’interface audio sur celle que vous voulez (il pourrait y en avoir plusieurs dans la liste)
  • Réglez la fréquence d’échantillonnage sur 48000
  • Paramétrez la trames/période à 128 et périodes/tampon à 2 dans un premier temps

Redémarrez JACK pour que les nouveaux paramètres soient pris en compte

Démarrez Jamulus

Une fois JACK lancé et configuré, démarrez Jamulus.

Si vous rencontrez des problèmes d’interruption du son (en particulier des décrochages signalés par JACK/QjackCtl), essayez de définir des valeurs plus élevées (par exemple, 256 trames ou 3 périodes). Des valeurs plus faibles (par exemple 64 trames) peuvent apporter de meilleures performances mais peut-être plus de problèmes de son. Voir la page dépannage sinon.
