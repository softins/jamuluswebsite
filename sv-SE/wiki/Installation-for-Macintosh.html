Installation för macOS

Se till att du redan har läst sidan Getting Started.

Uppgradera? Du kanske vill back up your configuration först.

  1. Download Jamulus (Universal build)
    Spegel 2:SourceForge
  2. Installera Jamulus: Öppna den nedladdade .dmg-filen, godkänn licensen, dra och släpp varje ikon du ser i fönstret (Jamulus Client och Server) till din Programmapp. Efter det kan du stänga detta fönster.
  3. Kör Jamulus. Nu bör du kunna använda Jamulus precis som vilken annan applikation som helst.

Du kan ta bort mappen i nedladdningskatalogen som innehåller .dmg-filen och mata ut “Jamulus”-enheten på ditt skrivbord. De behövs inte längre.

Observera: Du kan hitta gamla (legacy) versioner som stödjer föråldrade operativsystem på GitHub release page.
